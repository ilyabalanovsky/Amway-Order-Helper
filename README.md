# Amway Order Helper

Десктоп-приложение для разбора текста заказа и формирования Excel-таблиц оплаты по групповым заказам Amway.

Проект работает полностью локально и не использует:
- внешние базы данных
- серверы
- API
- облачные сервисы

Стек проекта:
- Python 3.11+
- PySide6
- SQLite
- openpyxl

## Возможности

### Разбор заказа
- вставка цельного текста заказа в одно большое поле
- автоматическое определение строк участников
- разбор ФИО, сумм, скидок и итоговой строки
- поддержка чисел с пробелами как разделителями тысяч
- сравнение итогов из текста с итогами, рассчитанными внутри программы

### Работа со справочником
- локальный справочник партнёров и групп на SQLite
- определение группы только через справочник
- подсветка неизвестных партнёров
- блокировка экспорта, если у участника не задана группа
- пакетное добавление новых партнёров

### Генерация Excel
- создание нового `.xlsx` файла
- добавление нового листа в существующий `.xlsx`
- генерация формул для строк участников, групповых итогов и общего итога
- цветовое оформление колонок и итогов
- поддержка ручных полей, которые нужны для конкретного заказа

### История и настройки
- сохранение заказов в SQLite
- повторное открытие и редактирование заказов
- повторная генерация Excel по сохранённому заказу
- настройки отправителя, города, расходов, процентов доставки и поведения после экспорта

## Локальный запуск

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Запуск приложения:

```bash
python main.py
```

## Тесты

```bash
pytest -q
```

## Сборка через PyInstaller

Проект подготовлен к сборке в `onefile`.

Сборка:

```bash
python build.py
```

Или напрямую:

```bash
pyinstaller --noconfirm --clean order_helper.spec
```

Результат сборки будет в папке `dist/`.

## Автосборка через GitHub Actions

В проект уже добавлен workflow для GitHub Actions, который собирает:
- Windows-версию
- macOS-версию

Файл workflow:

```text
.github/workflows/build.yml
```

Можно:
- запускать сборку вручную из вкладки `Actions`
- автоматически выпускать релизы по тегам вида `v0.1.0`

## Где хранятся данные

SQLite-база и локальные данные сохраняются в пользовательской директории приложения, а не рядом с исполняемым файлом.

Типичные пути:
- Windows: `%APPDATA%/amway-order-helper`
- macOS: `~/Library/Application Support/amway-order-helper`
- Linux: `~/.local/share/amway-order-helper`

Это важно для `PyInstaller --onefile`, чтобы приложение корректно запускалось на новой машине.
