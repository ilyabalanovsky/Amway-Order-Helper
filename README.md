# Amway Order Helper

Amway Order Helper — локальное desktop-приложение для Windows и macOS, которое превращает JSON заказа из расширения в готовые Excel-файлы для оплаты, истории заказов и товарных сводок.

Проект сделан для реальной ежедневной работы:
- без облаков и внешних серверов
- без сетевых баз данных
- с локальным хранением в SQLite
- с генерацией `.xlsx`, которые удобно дорабатывать и открывать в Excel

## Что умеет приложение

- Загружает JSON заказа через drag&drop или выбор файла.
- Автоматически распознаёт участников заказа, суммы, скидки и итоги.
- Сверяет участников со справочником партнёров и групп.
- Подсвечивает неизвестных партнёров и не даёт экспортировать заказ, пока группы не назначены.
- Хранит историю заказов локально и позволяет открывать их повторно.
- Генерирует основную Excel-таблицу оплат с формулами, группами и итогами.
- Добавляет новый лист в уже существующий Excel-файл.
- Формирует отдельную товарную сводную таблицу по подкорзинам и товарам.
- Поддерживает ручные корректировки прямо в интерфейсе.

## Почему это удобно

Amway Order Helper убирает рутину из процесса обработки заказов:
- не нужно вручную разбирать большой заказ по строкам
- не нужно заново собирать группы участников
- не нужно каждый раз создавать Excel с нуля
- не нужно хранить данные в сторонних сервисах

В результате пользователь получает один понятный инструмент, который закрывает полный цикл:
`JSON заказа -> проверка -> справочник -> история -> Excel`


## Основные сценарии

### 1. Обработка заказа
Пользователь загружает JSON-файл заказа, приложение автоматически показывает участников, суммы, скидки и ошибки, если что-то не сошлось.

### 2. Генерация таблицы оплат
После проверки данных приложение создаёт новый `.xlsx` или добавляет новый лист в существующий файл, сохраняя формулы, структуру и рабочую логику расчётов.

### 3. Работа со справочником
Партнёры и группы ведутся локально. Можно быстро добавлять новых партнёров, массово назначать группы и редактировать данные прямо в таблице.

### 4. Товарная сводка
Отдельной кнопкой создаётся товарная таблица по подкорзинам: с покупателями, группами, товарами, количеством и комментариями из базы.

## Технологии

- Python 3.11+
- PySide6
- SQLite
- openpyxl
- pytest
- PyInstaller

## Быстрый старт

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Запуск приложения:

```bash
python main.py
```

Запуск тестов:

```bash
pytest -q
```

## Сборка приложения

Проект подготовлен к упаковке в один исполняемый файл.

Сборка:

```bash
python build.py
```

Или напрямую:

```bash
pyinstaller --noconfirm --clean order_helper.spec
```

Результат будет в папке `dist/`.

## Где хранятся данные

Приложение хранит SQLite-базу и локальные данные в пользовательской директории, а не рядом с исполняемым файлом.

Типичные пути:
- Windows: `%APPDATA%/amway-order-helper`
- macOS: `~/Library/Application Support/amway-order-helper`
- Linux: `~/.local/share/amway-order-helper`

Это позволяет приложению корректно работать после сборки через `PyInstaller --onefile` и на новой машине.

## Для кого этот проект

Этот проект подойдёт тем, кто:
- регулярно работает с групповыми заказами Amway
- ведёт локальный справочник партнёров
- хочет быстро получать готовые Excel-файлы без ручной сборки
- ценит автономную работу без облаков и подписок
